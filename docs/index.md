# Hearts of Iron IV Starter Documentation

Welcome to the Hearts of Iron IV beginner documentation.

This project uses a Docs-as-Code approach: the documentation is written in Markdown, stored in Git, checked automatically, and prepared for hosted publication with MkDocs.

Start with the [Beginner Guide](beginner-guide.md).

## What This Documentation Covers

- Choosing a beginner-friendly country.
- Building civilian and military factories.
- Researching practical technologies.
- Managing production lines.
- Organizing armies.
- Understanding supply.
- Using air power.
- Setting limited goals.
- Learning from defeat.
